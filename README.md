# schizophrenia-bot
 
In the .env file located at the root of the project directory specify DISCORD_TOKEN, CLIENT_ID and GUILD_ID each on the new line without spaces like so:

```
DISCORD_TOKEN=MTE...G2Q
CLIENT_ID=112...774
GUILD_ID=867...729
GREETING_CHANNEL_ID=724...113
```
